# Integrations

Marketplace, shipping, tracking, and AI integrations are adapter-based. Until credentials and approval are available, the UI uses manual fallbacks: copy listing content and confirm external IDs, attach carrier tracking manually, and use deterministic local insights.

The Superbuy extension should target the configured application import endpoint and request only required host permissions. Marketplace API access, EasyPost/Shippo, 17TRACK, and AI providers require their respective credentials and terms approval.
